> ⚠️ **Navigation 3 constraints** — enforced in every task that touches navigation code:
> - Backstack: `rememberNavBackStack(SavedStateConfiguration(...), HomeRoute)` — **NOT** `rememberNavController()`
> - Rendering: `NavDisplay(backStack) { route -> when(route) { … } }` — **NOT** `NavHost`
> - Active tab: `backStack.lastOrNull()` passed as `currentRoute: NavKey?` — **NOT** `NavBackStackEntry`
> - Tab selection: `BottomNavItem.entries.find { it.route == currentRoute }` — **NOT** `hasRoute()`
> - Single-top: `if (backStack.lastOrNull() != route) { backStack.clear(); backStack.add(route) }` — **NOT** `launchSingleTop`
