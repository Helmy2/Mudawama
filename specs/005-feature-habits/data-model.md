## Domain Models (`feature:habits:domain`)

### `Habit`

Immutable value type. Mutations produce a new instance passed to `upsertHabit`.

| Field | Type | Nullable | Notes |
|-------|------|----------|-------|
| `id` | `String` | No | UUID v4, generated by `CreateHabitUseCase` via `kotlin.uuid.Uuid.random()`. |
| `name` | `String` | No | Non-blank; enforced by `CreateHabitUseCase` / `UpdateHabitUseCase`. |
| `iconKey` | `String` | No | Token resolving to a designsystem icon; falls back to placeholder on unknown key. |
| `type` | `HabitType` | No | `BOOLEAN` or `NUMERIC`. |
| `category` | `String` | No | Grouping label; defaults to `"custom"` for user-created habits. |
| `frequencyDays` | `Set<DayOfWeek>` | No | At least one day required. Uses `kotlinx.datetime.DayOfWeek`. |
| `isCore` | `Boolean` | No | `true` = seeded ritual; deletion blocked by `DeleteHabitUseCase`. |
| `goalCount` | `Int?` | Yes | Target count for NUMERIC habits. Always `null` for BOOLEAN habits. |
| `createdAt` | `Long` | No | Epoch ms; set once at creation. |

**Invariants**: `name.isNotBlank()` · `frequencyDays.isNotEmpty()` · BOOLEAN implies `goalCount == null`

---

### `HabitLog`

One record per `(habitId, date)` pair.

| Field | Type | Nullable | Notes |
|-------|------|----------|-------|
| `id` | `String` | No | UUID v4; assigned on first creation, reused on updates. |
| `habitId` | `String` | No | FK to `Habit.id`. Room ForeignKey with CASCADE delete. |
| `date` | `String` | No | ISO-8601 `yyyy-MM-dd`; produced by `toIsoDateString(TimeProvider.logicalDate())`. |
| `status` | `LogStatus` | No | `PENDING` or `COMPLETED`. |
| `completedCount` | `Int` | No | Increment target for NUMERIC habits; always `0` for BOOLEAN. |
| `loggedAt` | `Long` | No | Epoch ms of last write; updated on every upsert. |

---

### `HabitWithStatus`

Read-only projection emitted by `ObserveHabitsWithTodayStatusUseCase`. Never persisted.

| Field | Type | Nullable | Notes |
|-------|------|----------|-------|
| `habit` | `Habit` | No | Full habit definition. |
| `todayLog` | `HabitLog?` | Yes | Log for today's logical date. `null` = habit not touched today. |
| `weekLogs` | `List<HabitLog?>` | No | Exactly 7 entries. `weekLogs[0]` = today, `weekLogs[6]` = 6 days ago. `null` entry = no log for that day. |

**Derived display properties** (computed in composables, never stored):

| Property | Expression |
|----------|-----------|
| `isCompletedToday` | `todayLog?.status == LogStatus.COMPLETED` |
| `isDueToday` | today's `DayOfWeek` in `habit.frequencyDays` |
| `numericProgress` | `todayLog?.completedCount ?: 0` |
| `isNumericGoalReached` | `habit.goalCount != null && numericProgress >= habit.goalCount` |

### `BottomSheetMode`

```kotlin
sealed interface BottomSheetMode {
    data object Hidden                             : BottomSheetMode
    data object AddHabit                           : BottomSheetMode
    data class  OptionsMenu(val habit: Habit)      : BottomSheetMode
    data class  EditHabit(val habit: Habit)        : BottomSheetMode
    data class  DeleteConfirm(val habitId: String) : BottomSheetMode
}
```

State transitions:

| From | Event | To |
|------|-------|----|
| `Hidden` | FAB tapped | `AddHabit` |
| `Hidden` | Habit long-pressed | `OptionsMenu(habit)` |
| `OptionsMenu` | Edit tapped | `EditHabit(habit)` |
| `OptionsMenu` | Delete tapped | `DeleteConfirm(habitId)` |
| `AddHabit` or `EditHabit` | Save (valid) | `Hidden` + create/update effect |
| `DeleteConfirm` | Confirm Delete | `Hidden` + delete effect |
| any visible | dismiss or Cancel | `Hidden` |

## Entity to Domain Mapper Summary

### `HabitEntity` <-> `Habit`

| Entity column | Domain field | Conversion |
|---|---|---|
| `type: String` | `type: HabitType` | `HabitType.valueOf(entity.type)` / `habit.type.name` |
| `frequencyDays: String` | `frequencyDays: Set<DayOfWeek>` | split by `,`, map each to `DayOfWeek(it.toInt())` / join with `isoDayNumber.toString()` |
| All others | All others | Identity |

### `HabitLogEntity` <-> `HabitLog`

| Entity column | Domain field | Conversion |
|---|---|---|
| `status: String` | `status: LogStatus` | `LogStatus.valueOf(entity.status)` / `log.status.name` |
| All others | All others | Identity |
